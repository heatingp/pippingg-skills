# pippingg-skills

帮助深度思考、内容创作和菜谱生成的 Claude Code 技能包。

[English](README.md)

## 安装方式

**方式一：添加 Marketplace（推荐）**

```bash
/plugin marketplace add heatingp/pippingg-skills
```

**方式二：直接安装**

```bash
/plugin install thinking-skills@pippingg-skills
/plugin install content-skills@pippingg-skills
```

## 技能列表

### 1. socratic-thinking（苏格拉底式思考）

一个帮你深度思考并写出文章的苏格拉底式提问伙伴。

**核心功能：**
- 用第一性原理挑战你的假设
- 区分真问题和情绪驱动的假问题
- 帮你跳出思维框架
- 引导你从零散想法到结构化文章

**特色能力：**
- **边界检查**：质疑之前先问清上下文（避免抬杠）
- **第一性原理**：回到问题根源，不在表层打转
- **反向思考**："如果完全相反呢？"
- **穿透式提问**：追踪钱的流向、识别受益者、揭示隐藏利益
- **搭梯子**：当你卡住时，提供更小的台阶而非直接给答案

**避免的反模式：**
- 当"舔狗"，什么都同意
- 给 400 字的长篇大论而不是提问
- 没问清前提就质疑结论
- 在你的框架里打转，而不是帮你跳出来

**使用方式：**

```bash
/socratic-thinking
```

或者直接说："帮我想想"、"我想讨论"、"帮我分析一下"

### 2. skill-iterator（技能迭代器）

根据对话反馈自动迭代更新技能。

**核心功能：**
- 从对话中提取反馈
- 追踪「反馈 → 修正 → 确认」循环
- 生成 diff 预览后再应用修改
- 维护 CHANGELOG 记录技能演进

**命令：**

```bash
/skill-iterator              # 更新当前对话使用的技能
/skill-iterator <skill-name> # 更新指定技能
/feedback <内容>             # 显式标记重要反馈（优先处理）
```

**工作流程：**
1. 识别目标技能
2. 读取 SKILL.md 和 CHANGELOG.md
3. 分析对话历史，提取反馈循环
4. 生成修改建议和 diff 预览
5. 用户确认后应用修改

**智能建议：**
- 检测到同一部分被改了 3+ 次 → 建议增加专门章节
- 检测到改过又改回来 → 提醒确认
- 检测到某类反馈占比过高 → 建议重构

**使用方式：**

使用任意技能并在对话中提供反馈后：

```bash
/skill-iterator
```

### 3. recipe-generator（菜谱生成器）

生成专业级详细菜谱，附带生图提示词和自动生成的菜谱海报。

**核心功能：**

- 检索权威来源（米其林指南、舌尖上的中国、专业大厨）
- 生成精确配方（克、毫升等精确用量）
- 创建多种风格的菜谱海报生图提示词
- 自动使用 Gemini 生成菜谱海报图片

**特色能力：**

- **权威来源**：整合米其林、舌尖上的中国、高分菜谱
- **精确用量**：告别"适量"、"少许"——精确到克和毫升
- **大厨技巧**：真正实用的非显而易见的专业技巧
- **自动生图**：自动生成精美的菜谱海报

**风格选项：**

| 风格 | 说明 |
| ---- | ---- |
| C（默认） | 现代写实风，教育感强 |
| A | 传统海报风，视觉冲击强 |
| B | 文人卷轴风，素雅内敛 |

**使用方式：**

```bash
/recipe-generator 清蒸鱼
/recipe-generator 红烧肉 plan A风格
```

**依赖：**

- `baoyu-gemini-web` — 用于生成菜谱海报图片

## 为什么需要这些技能？

**问题：** AI 助手容易变成"舔狗"——什么都同意、只做表面分析、在你的思维框架里打转。

**解决方案：** 这些技能训练 Claude 成为真正的思考伙伴：
- 挑战你的想法（但先问清上下文）
- 用苏格拉底式提问而非说教
- 帮你自己发现洞察
- 通过对话反馈持续改进
